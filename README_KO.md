## ✨ 주요 기능

- 🎯 **스마트 강조** - 고품질 언어 콘텐츠를 자동으로 식별하고 강조
- 💬 **호버 카드** - 즉시 정의, 발음 및 예문 표시
- ⭐ **컬렉션** - 학습 자료를 저장하고 정리
- ☁️ **클라우드 동기화** - 모든 기기에서 컬렉션 액세스
- 🌍 **다국어 UI** - 简体中文, English
- 📊 **학습 모드** - Lite, Standard, Detailed (수준에 맞게 조정)
- 🎓 **개인화** - 학습 중점 영역 및 목표 맞춤 설정

## 📦 설치 방법

### 옵션 1: 공식 스토어 (권장)
**⏳ 현재 심사 중 (7일 예상)**

- **Chrome 웹 스토어**: 곧 출시
- **Edge 확장 프로그램**: 곧 출시

### 옵션 2: 수동 설치 (지금 사용 가능)

#### Chrome / Edge / Brave 브라우저:

1. **다운로드** 최신 버전:
   - [Releases 페이지](../../releases) 방문
   - `lemingle-v1.0.2.zip` 다운로드

2. **압축 해제** zip 파일을 폴더에 압축 해제

3. **확장 프로그램 설치**:
   - 브라우저를 열고 `chrome://extensions` 또는 `edge://extensions` 방문
   - 오른쪽 상단의 **개발자 모드** 활성화
   - **압축해제된 확장 프로그램을 로드합니다** 클릭
   - 압축 해제된 폴더 선택
   - 완료! 🎉

4. **고정** 쉬운 액세스를 위해 도구 모음에 확장 프로그램 고정

## 🚀 빠른 시작

1. **웹페이지 방문** 영어 콘텐츠가 있는 페이지 (예: 뉴스, 기사, 블로그)
2. **강조 표시 확인** 고품질 언어 콘텐츠에 자동으로 표시
3. **마우스 오버** 강조 표시 위에 마우스를 올려 정의 및 발음 확인
4. **⭐ 클릭** 컬렉션에 항목 저장
5. **확장 프로그램 아이콘 클릭** 컬렉션 보기 및 설정 조정

## 🔧 설정

확장 프로그램 아이콘 클릭 → ⚙️ 설정에서 맞춤 설정:

- **학습 모드**: 강조 밀도 제어
- **집중 콘텐츠**: 카테고리 선택 (연어, 관용구, 연결어)
- **UI 언어**: 인터페이스 언어 전환
- **학습자 프로필**: 수준, 목표 및 선호도 설정
