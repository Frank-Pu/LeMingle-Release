> 將任何網頁變成你的個人語言學習材料。智能高亮優質詞彙、搭配、習語和連接詞，即時顯示釋義和發音。

## ✨ 功能特點

- 🎯 **智能高亮** - 自動識別並高亮優質語言內容
- 💬 **懸浮卡片** - 即時顯示釋義、發音和例句
- ⭐ **收藏管理** - 保存和整理你的學習材料
- ☁️ **雲端同步** - 在所有裝置上訪問你的收藏
- 🌍 **多語言介面** - 支援簡體中文、English
- 📊 **學習模式** - 精簡、標準、詳細（根據你的水平調整）
- 🎓 **個性化** - 自訂學習重點和目標

## 📦 安裝方法

### 方式 1：官方商店（推薦）
**⏳ 審核中（預計 7 天）**

- **Chrome 線上應用程式商店**：即將上線
- **Edge 擴充功能商店**：即將上線

### 方式 2：手動安裝（立即可用）

#### Chrome / Edge / Brave 瀏覽器：

1. **下載**最新版本：
   - 訪問 [Releases 頁面](../../releases)
   - 下載 `lemingle-v1.0.3.zip`

2. **解壓縮** zip 檔案到一個資料夾

3. **安裝**擴充功能：
   - 開啟瀏覽器，訪問 `chrome://extensions` 或 `edge://extensions`
   - 啟用右上角的**開發人員模式**
   - 點擊**載入未封裝項目**
   - 選擇解壓縮後的資料夾
   - 完成！🎉

4. **釘選**擴充功能到工具列，方便使用

## 🚀 快速開始

1. **訪問任何網頁**，包含英語內容（如新聞、文章、部落格）
2. **查看高亮**，自動出現在優質語言內容上
3. **懸停**在任何高亮上，查看釋義和發音
4. **點擊 ⭐** 保存項目到收藏
5. **點擊擴充功能圖示**查看收藏並調整設定

## 🔧 設定選項

點擊擴充功能圖示 → ⚙️ 設定來自訂：

- **學習模式**：控制高亮密度
- **關注內容**：選擇類別（搭配、習語、連接詞）
- **介面語言**：切換介面語言
- **學習檔案**：設定你的水平、目標和偏好
