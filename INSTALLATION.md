#### Option 3: Manual Installation (Available Now)

1. **Download the Extension**
   - Go to [Releases](../../releases/latest)
   - Download `lemingle-v1.0.2.zip`
   - Save to your computer

2. **Extract the ZIP File**
   - Right-click the downloaded file
   - Select "Extract All..." or use your preferred extraction tool
   - Remember the extraction location

3. **Open Extensions Page**
   - **Chrome**: Navigate to `chrome://extensions`
   - **Edge**: Navigate to `edge://extensions`
   - **Brave**: Navigate to `brave://extensions`

4. **Enable Developer Mode**
   - Look for "Developer mode" toggle in the top-right corner
   - Turn it ON

5. **Load the Extension**
   - Click "Load unpacked" button
   - Navigate to the extracted folder
   - Select the folder and click "Select Folder"

6. **Pin the Extension** (Recommended)
   - Click the puzzle icon in your browser toolbar
   - Find "LeMingle" in the list
   - Click the pin icon to keep it visible

7. **Start Using!** ğŸ‰
   - Visit any English webpage
   - See smart highlights appear automatically
   - Hover over highlights for definitions
   - Click the extension icon to manage collections

### Troubleshooting

**Extension doesn't appear after installation**
- Refresh the extensions page
- Make sure Developer mode is enabled
- Try restarting your browser

**Highlights not showing**
- Click the extension icon
- Check if the extension is enabled
- Verify the current website is not in the disabled list

**Login issues**
- Check your internet connection
- Clear browser cache and try again
- Make sure you're using the correct email
