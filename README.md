# 🌐 LeMingle - 智能语言学习助手

[![Chrome 网上应用店](https://img.shields.io/badge/Chrome-安装-blue?logo=google-chrome)](https://chrome.google.com/webstore)
[![Edge 扩展商店](https://img.shields.io/badge/Edge-安装-blue?logo=microsoft-edge)](https://microsoftedge.microsoft.com/addons)

> 将任何网页变成你的个人语言学习材料。智能高亮优质词汇、搭配、习语和连接词，即时显示释义和发音。

**[English](README_EN.md) | 简体中文 | [繁體中文](README_ZH_TW.md) | [한국어](README_KO.md) | [Français](README_FR.md)**

---

## ✨ 功能特点

- 🎯 **智能高亮** - 自动识别并高亮优质语言内容
- 💬 **悬浮卡片** - 即时显示释义、发音和例句
- ⭐ **收藏管理** - 保存和整理你的学习材料
- ☁️ **云端同步** - 在所有设备上访问你的收藏
- 🌍 **多语言界面** - 支持简体中文、English
- 📊 **学习模式** - 精简、标准、详细（根据你的水平调整）
- 🎓 **个性化** - 自定义学习重点和目标

## 📦 安装方法

### 方式 1：官方商店（推荐）
**⏳ 审核中（预计 7 天）**

- **Chrome 网上应用店**：即将上线
- **Edge 扩展商店**：即将上线

### 方式 2：手动安装（立即可用）

#### Chrome / Edge / Brave 浏览器：

1. **下载**最新版本：
   - [下载插件压缩包 (lemingle-v1.0.2.zip)](./lemingle-v1.0.2.zip)

2. **解压** zip 文件到一个文件夹

3. **安装**扩展：
   - 打开浏览器，访问 `chrome://extensions` 或 `edge://extensions`
   - 启用右上角的**开发者模式**
   - 点击**加载已解压的扩展程序**
   - 选择解压后的文件夹
   - 完成！🎉

4. **固定**扩展到工具栏，方便使用

## 🚀 快速开始

1. **访问任何网页**，包含英语内容（如新闻、文章、博客）
2. **查看高亮**，自动出现在优质语言内容上
3. **悬停**在任何高亮上，查看释义和发音
4. **点击 ⭐** 保存项目到收藏
5. **点击扩展图标**查看收藏并调整设置

## 🔧 设置选项

点击扩展图标 → ⚙️ 设置来自定义：

- **学习模式**：控制高亮密度
- **关注内容**：选择类别（搭配、习语、连接词）
- **界面语言**：切换界面语言
- **学习档案**：设置你的水平、目标和偏好
